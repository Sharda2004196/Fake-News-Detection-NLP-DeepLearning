# Fake News Detection using NLP and Deep Learning

This project implements a fake news detection system using NLP and deep learning techniques.

**Tech Stack:** Python, TensorFlow, Scikit-learn  
**Model:** Embedding → Conv1D → BiLSTM  
**Accuracy:** ~81%
