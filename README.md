# Fake News Detection using NLP and Deep Learning

This project implements a fake news detection system using NLP and deep learning techniques with a Bi-LSTM + Conv1D architecture

**Tech Stack:** Python, TensorFlow, Scikit-learn, NLTK

**Model:** Embedding → Conv1D → BiLSTM 

**Accuracy:** ~81%
